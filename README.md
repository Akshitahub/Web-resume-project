## Overview
Personal portfolio showcasing skills, projects, internships, education, and contact info. Responsive design with dark/light mode and smooth animations.

## Tech
- HTML, TailwindCSS, JavaScript  
- Font Awesome, AOS

## Features
- Hero section with gradient text and typing effect  
- Downloadable CV and contact links  
- Projects and skill badges with hover effects  
- Dark mode toggle and smooth scroll
